# ICMP Ping Test вҖ” veth Pair

> **Scope:** ICMP echo request/reply over virtual Ethernet using AF_PACKET or AF_XDP PMDs.
> No VM required вҖ” runs entirely in host namespaces.

---

## At a Glance

| | |
|---|---|
| **Script** | `tests/ping_veth.sh` |
| **Transport** | veth pair (AF_PACKET / AF_XDP) |
| **Peer** | Alpine container (kernel ICMP responder) |
| **Modes** | Interval (5 pings) В· Flood (line-rate N seconds) |
| **Pass** | All replies received (interval) В· tx_pkts > 0 (flood) |
| **Code exercised** | `icmp.c` В· `ipv4.c` В· `arp.c` В· `ethernet.c` В· `port_init.c` |

### Interface Names

| Mode | Host-side veth | Peer-side veth | DPDK vdev |
|------|---------------|----------------|-----------|
| AF_PACKET | `veth-vaigai` | `veth-tpeer` | `net_af_packet0,iface=veth-vaigai` |
| AF_XDP | `veth-xdp` | `veth-xdppeer` | `net_af_xdp0,iface=veth-xdp` |

### Test Modes

**Interval** (default) вҖ” Sends 5 ICMP echo requests, 56 bytes each, 1 s apart.
Pass criterion: all 5 replies received, 0% loss.

**Flood** (`--flood N`) вҖ” Sends ICMP at line rate for N seconds.
Pass criterion: transmitted packet count > 0. Prints throughput (pps) on completion.

### Usage

```bash
# AF_PACKET вҖ” interval
bash tests/ping_veth.sh

# AF_PACKET вҖ” flood 5s
bash tests/ping_veth.sh --flood 5

# AF_XDP вҖ” interval
bash tests/ping_veth.sh --xdp

# AF_XDP вҖ” flood 5s
bash tests/ping_veth.sh --xdp --flood 5
```
